JALES (Just Another LEarning Software.)
#Version 1.0 - First Release 2025 May 22

We are excited to announce the official launch of our software application!
This first release includes the basic features for your LEARNING experience.
The main idea behind this software is that the "Board Creator" can introduce their board with a "How to Use It" guide.
With an intuitive user interface and fast performance, this release is designed to provide you with a seamless experience.
We look forward to your feedback and are committed to improving the app with future updates.

## Installation
Currently this application only tested and support Windows OS.
To install and use "JALES" apps, follow the steps below:
1. Download the latest release from the [[[Releases page]]].
2. Unzip the downloaded file in any drive (Recomended not inside C:\).
3. You can RUN by click "JALES.exe" file.
4. Installation DONE.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
## Usage
After installation, launch the application and you'll be greeted with the main Form/Page.
There are 2 sections, 

For End Users :
1. Click the "Open File" button.
2. Select file with extention *.pzJales.
3. And have a good learning experience.

For Board Creator :
To get better experience you can follow tutorial on youtube.
Link : https://www.youtube.com/@WAH.Learning.
